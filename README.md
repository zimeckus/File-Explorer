# Fast File Explorer
A file explorer written in Rust

## Supported operating systems
- Linux


## Steps
```
#  Make sure you have Tauri CLI installed
cargo install tauri-cli

# Install dependencies
yarn

# Run app for development
cargo tauri dev

# Build for production
cargo tauri build
```
